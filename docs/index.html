Reddit Visuals

Retrieved, cleaned, and analyzed approximately 1.3 billion data points over 20,000+ posts on Reddit to identify several major trends and produced 28 data visualizations to effectively represent them

Identify Coral Health Changes

A computer vision solution for the MATE Remotely Operated Vehicle challenge. Identifies changes in coral health from pictures taken by the Columbia University Robotics Club rover.

Escape From the Underworld

3D Unity Platformer game coded in C# and built with the Unity Game Engine. This single-level game is used as a demonstration of core concepts in Juni Learning's Game Developement in Unity course which I created.

Doodle Jump in Python with Turtle

Recreated Doodle Jump with animations, physics, skins, menus, and more solely using the minimal Turtle graphics library.

Gmail Compiler

A personal use script that I use to compile my daily school related emails into a readable webpage. It helps manage frequently recieved emails by making mass access and mark-as-read simple.

Sorting Visualizations in PyGame

Built funadmental sorting algorithms including Bubble Sort, Selction Sort, and Insertion Sort into a GUI using PyGame.

Intro to Unity Game Development and C#

Wrote a course teaching the fundamentals of C# and Unity Game design launched by Juni Learning in 2022. The course is taught by dozens of Juni instructors and features projects and challenges along with learning resources and C# solution code.
