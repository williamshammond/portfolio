Coral Health Identifier

A computer vision project for Columbia Univeristy Robotics Club

The Task

My subteam's task for this project was to identify coral growth, death, bleaching, and recovery based on simulated coral colonies in a pool environment. The program we created will be used to Identify the changes in a simulated coral colony from a before image and an after image that will eventually be taken by our team's remotely operated underwater vehicle.

The Solution

We used OpenCV to separate various features out from both the before and after images with specified HSV ranges. We alligned the two images using feature based image allignment. We found differences between alligned features and created homogenous "blocks" of our difference values to identify clear areas of change. Finally, we made use of logical operations to identify specific features such as growth and death. Due to our use of HSV image processing as well as the "blocking" step of the program, the process is robust and deals with the distortion and image tint that will accompany the images that will eventually be taken by our rover.
