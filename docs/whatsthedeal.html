What's the Deal?

Real estate information for NYC.

Site Purpose

As a real estate enthusiast myself, I wanted to design a website that could provide information that I was desperately missing in my own search for better knowledge. As I have pursued opportunities to participate in the New York real estate market, both as an analyst and a personal investor, I found it extremely hard to find information as simple as "Who financed this project?" "How much did it cost?" or "Who are the most active developers in the city>"

I set out to fix that problem by compiling basic information on properties and deals throughout the New York City area and displaying them on an interactive and easily searchable website.

What's The Deal?

The site offers featured deals as well as a database searchable by project names, developers, neighborhoods, and more. The site is not yet actively hosted.

Maps API

Each data point offers an interactive map as one of its key features. This helps interested users locate the project visually and get a better picture of the surrounding businesses and amenities. The Google Maps API was critical to this feature as it allows for easily and dynamically constructed embeded maps on each page.
