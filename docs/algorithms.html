Sorting Visualizations in Pygame

Graphical visualizations of simple sorting techniques to help teach beginners.

Inspiration

I was inspired to make an easily accesible GUI visualization of simple sorting algorithms by my time teaching such algorithms to beginners at Juni Learning. I was particularly inspired by the fantastic website visualgo.com which allows learners to interact with algorithms as they progress, slow them down, and reverse them as nececesary.

Implementation

Implementing algoritms such as these by themselves is quite the trivial task, of course. However, it was a bit more fun adding the GUI aspect of this program using Pygame to make the algorithms visual and interactive. I built manual animations into each algorithm so that the steps and logic could easily be seen by a beginner and so that they could be slowed down or sped up as necesary.
